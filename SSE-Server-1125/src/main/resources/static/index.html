SSE / WebSocket 测点推送示例

页面可分别控制单连接每秒推送的测点数量 `countOfPoint` 与并发连接数 `countOfConn`。服务端每秒生成一组随机整数测点， 分别通过 SSE 和 WebSocket 推送到当前页面；同时展示服务端与浏览器的 CPU / 内存情况，便于比较两种协议在连接数变化下的内存影响。

说明：浏览器内存依赖 `performance.memory`，仅部分 Chromium 内核浏览器支持；浏览器 CPU 为基于事件循环空闲度的近似值。
